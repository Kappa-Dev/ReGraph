Find an instance of a pattern in a specified graph.

graph_id – id of a graph in the hierarchy to search for matches; pattern – nx.(Di)Graph object defining a pattern to match; pattern_typing – a dictionary that specifies a typing of a pattern, keys of the dictionary – graph id that types a pattern, this graph should be among parents of the graph_id graph; values are mappings of nodes from pattern to the typing graph;
